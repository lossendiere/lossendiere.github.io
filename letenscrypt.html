Renouveller les certificat Letenscrypt d'un serveur

Arréter le serveur Apache : /etc/init.d/apache2 stop

Renouveller les certificats du serveur : certbot-auto renew

Si tous se passe bien message : « Congratulations, all renewals succedeed »

Vérifier que les certificats sont renouvellés pour 90 jours : certbot-auto certificates

redémarrer le serveur apache : /etc/init.d/apache2 start


Ajouter une tache Cron pour un renouvellement automatique
