Site personnel 

Créer des sites web avec un compte Github

1 ) Créer le premier repository

Il faut dans le nom du repository remettre le nom de votre login suivi de github.io

------------------------------------------------------------------------------------

2 ) Vérifier la publication du repository principal

Si le site n'est pas publié, il faudra probablement créer une branche "gh-pages" et paramétrer cette branche comme github pages repository

3 ) Créer les repository pour les sites internet

Ne pas oublier d'initialiser le repository avec un premier fichier README, sinon le repository ne pourra pas être paramétré comme github pages

--------------------------------------------------------------

4 ) Création d'une deuxième branche

Il faut créer une deuxième branche du repository réservé à la publication des pages web.
Cette branche doit être nommée "gh-pages"

5 ) Vérifiez la source de Github

Vérifiez que c'est bien "gh-pages"

6 ) Eventuellement changer la branche par défaut

La branche master c'est celle dont le contenu s'affiche par défaut quand on va dans le repository mais la branche active est bien gh-pages
Vous changer la branche par défaut si vous le désirez.

-----------------------------------------------------------
